============================================================
特征重要性统计分析报告
============================================================

1. 数据概览
   原始数据文件: data/235UALL.csv
   样本数量: 4127
   原始特征数: 4 (Z, A, E, Yield)
   新增特征数: 15
   总特征数: 20

2. 关键发现
   2.1 与Yield相关性最强的特征 (Spearman |ρ|):
      1. N_over_Z            : |ρ| = 0.174 (p < 0.001 ***)
      2. symmetry_energy     : |ρ| = 0.114 (p < 0.001 ***)
      3. mass_excess         : |ρ| = 0.108 (p < 0.001 ***)
      4. Z_magic_dist        : |ρ| = 0.105 (p < 0.001 ***)
      5. Z_magic_prox        : |ρ| = 0.105 (p < 0.001 ***)

   2.2 奇偶性效应:
      OE: 4127样本, 平均Yield = 5.601e-02

   2.3 幻数效应:
      壳闭合核: 2741样本, 平均Yield = 4.031e-02
      非壳闭合核: 1386样本, 平均Yield = 8.705e-02

3. 建议
   基于统计分析，建议在机器学习模型中优先考虑以下特征:
     1. N_over_Z
     2. symmetry_energy
     3. mass_excess
     4. Z_magic_dist
     5. Z_magic_prox

   对于分类特征，建议进行独热编码或目标编码。
   对于连续特征，注意检查与现有特征(Z, A, E)的多重共线性。
