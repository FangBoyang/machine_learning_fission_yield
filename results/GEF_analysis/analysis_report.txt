============================================================
GEF 特征重要性统计分析报告
============================================================

1. 数据概览
   原始数据文件: data/GEF.csv
   样本数量: 18267
   原始特征数: 4 (Z, A, E, Yield)
   新增特征数: 15
   总特征数: 19

2. 关键发现
   2.1 与Yield相关性最强的特征 (Spearman |ρ|):
      1. E                   : |ρ| = 0.997 (p < 0.001 ***)
      2. symmetry_energy     : |ρ| = 0.154 (p < 0.001 ***)
      3. N_magic_dist        : |ρ| = 0.065 (p < 0.001 ***)
      4. N_magic_prox        : |ρ| = 0.065 (p < 0.001 ***)
      5. N                   : |ρ| = 0.065 (p < 0.001 ***)

   2.2 奇偶性效应:
      OE: 18267样本, 平均Yield = 9.282e-03

   2.3 幻数效应:
      壳闭合核: 18267样本, 平均Yield = 9.282e-03
      非壳闭合核: 0样本, 平均Yield = nan

3. 建议
   基于统计分析，建议在机器学习模型中优先考虑以下特征:
     1. E
     2. symmetry_energy
     3. N_magic_dist
     4. N_magic_prox
     5. N

   对于分类特征，建议进行独热编码或目标编码。
   对于连续特征，注意检查与现有特征(Z, A, E)的多重共线性。
